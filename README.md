# Music Player UI (HTML & CSS)

## Project Overview

This project is a **simple and visually pleasing Music Player User Interface** created using **HTML and CSS only**.
The design uses a **soft pink and purple gradient background** with a **matching glass-style music player card**.
It focuses on **clean layout, color harmony, and beginner-friendly code**.

---

## Features

* Built using **pure HTML and CSS**
* **Pink–purple gradient background**
* Glassmorphism-style music player card
* Rectangular album display (no circular design)
* Progress bar to represent playback
* Basic control buttons (Previous, Play, Next)
* Simple and clean UI suitable for beginners

---

## Technologies Used

* **HTML5** – Structure of the music player
* **CSS3** – Styling, layout, gradients, and effects
